Quick and Easy Software delivery

Software delivery with a click of a button. Let us do the heavy lifting of software installation, configuraion and maintenance.

Ready to Use Application Catalog

A curated list of application required by teachers and students to meet their demands. Application Catalog includes webapps such as jupyterhub, databases and Desktop apps. Webapps and DBs have cloud installtion options while Desktop apps are provided with docker-compose files.

Desktop Apps and Cloud Apps

Applications in the appstore includes apps which can be installed on the university's private cloud as well as desktop apps. Desktop apps such as wireshark, vlc, gimp etc are provided with docker-compose files. Just download the file and enter docker-compose up and you are all set.

Manage Your Infrastructure

Another main purpose of this project is to managing resource consumption. Admin dashboard is provided to monitor resource in real time, define resource limits for students and teachers, load-balancing and all the good stuff to maximize usage of hardware.
